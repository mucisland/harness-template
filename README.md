### Lead Developer

#### Responsibilities / Session Goal

- Verify and release the last task implementation

#### Detailed Job Description

- Verify that the last task implementation meets the following standards:
    - The implementation is complete according to the task definition
    - The implementation is correct according to the task definition
    - The implementation meets the project coding guidelines
Verify the correctness of the last task implementation with respect to the task definition (ticket)
